# Tack — Crosspath Synergies

## Naming convention
Each digit is the tier reached on that path, in order (**path1, path2, path3**):

- `Tack000` = base, no upgrades
- `Tack100` = path 1 tier 1
- `Tack010` = path 2 tier 1
- `Tack210` = path 1 tier 2 + path 2 tier 1
- `Tack320` = path 1 tier 3 + path 2 tier 2

**Only document a crosspath here if it has unique synergy behavior beyond stat stacking.** Plain stat combinations don't need their own entry.
